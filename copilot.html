Tentang halaman ini. Repo fantasy-world-cup-skill (github.com/manavm12/fantasy-world-cup-skill) bukan library JS atau API web — ini adalah skill untuk AI agent (instruksi di SKILL.md + data lokal + CLI Tinyfish untuk riset langsung dari web). Tidak bisa ditempel sebagai tombol di halaman statis tanpa backend yang menjalankan LLM + CLI.

Yang bisa dan sudah diintegrasikan ke app ini:

  • Aturan resmi lengkap dari references/game-rules.md milik skill itu — dirender di bawah (budget, komposisi skuad, limit pemain per negara per babak, transfer, booster, kapten, substitusi).
  • data/rounds.json — sumber fixture resmi FIFA (play.fifa.com/json/fantasy/rounds.json) yang dipakai skill ini, sekarang juga di-fetch oleh scraper.js kita.

Fitur chat-nya ("build my team", "compare X vs Y", dst.) tetap bisa dipakai — tapi lewat percakapan dengan AI agent (Claude Code) langsung, bukan dari halaman ini. Lihat panduan di bawah.

Status Data & Tools

Cara Pakai Copilot-nya

Buka percakapan dengan AI agent kamu (Claude Code) di repo fantasy-world-cup-skill yang sudah di-clone, lalu tanya seperti contoh berikut (sesuai README skill-nya):

help me build my team
build from scratch with 100m and aggressive picks
best cheap defenders for the next round
compare Lautaro Martinez vs Isak
give me a safer version of this squad

Agent akan baca data/players.json/squads.json/rounds.json lokal untuk fakta stabil (harga, posisi, status, fixture), lalu pakai Tinyfish search/fetch untuk hal yang butuh info terbaru (starting XI, cedera, suspensi, preview line-up).

Aturan Skuad & Budget

Budget awal: 100.0m · Ukuran skuad: 15 pemain · Komposisi: 2 GK – 5 DEF – 5 MID – 3 FWD (sama dengan yang dipakai di semua halaman Best 15 app ini).

Formasi starting XI yang valid: 4-4-2, 4-3-3, 4-5-1, 3-4-3, 3-5-2, 5-4-1, 5-3-2.

Budget bertambah +5.0m begitu fase grup selesai (Round 3 lock) dan transfer Round of 32 dibuka.

Kapten & Substitusi

  • Kapten dapat 2× poin. Kalau kapten main 0 menit, wakil kapten otomatis dapat 2× — kecuali user sudah melakukan perubahan manual saat ronde live.
  • Saat ronde live, kapten bisa diganti berulang kali, asal pemain baru belum main dan pertandingan kapten lama sudah selesai.
  • Sebelum lockout, starting XI/bench/kapten/wakil kapten bisa diubah bebas asal formasi tetap valid.
  • Pemain bench tetap dapat poin, tapi tidak terhitung kecuali disubstitusi masuk (manual atau otomatis).
  • Auto-substitusi hanya terjadi kalau user tidak melakukan perubahan manual apa pun saat ronde live.

Skor (ringkas — detail lengkap di halaman Daftar Pemain saat expand baris pemain)

Aturan ini sudah diimplementasikan persis di breakdown poin halaman Daftar Pemain: Appearance, Assist (+3), Yellow/Red Card, Own Goal, Penalty Won/Conceded, Goal Scored & Clean Sheet (bobot beda per posisi), Saves/Tackles/Chances Created/Shots on Target bonus, plus bonus gol free-kick langsung (+1) dan scouting bonus (+2 kalau >4 poin & dipilih <5% tim).
